PulseExtenderMod is a simple plugin that allows you to set the duration 
a button is pressed in Bukkit.
    
TO INSTALL

    Copy the PulseExtenderMod.jar file to the Bukkit plugin folder.


PERMISSIONS

    pulseextender.admin: Allows altering the global button length setting.

    
HOW TO COMPILE

    If it's not already referenced, include CraftBukkit 1.3.1 R2.0:
        craftbukkit-1.3.1-R2.0.jar
